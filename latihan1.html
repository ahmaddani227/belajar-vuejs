    template

    Kita bisa mengelompokkan beberapa elemen sesuai kondisi atau syaratnya. kita tau directive v-if harus ditempelkan ke elemen parent. Biasanya saya tambahkan di elemen div. Tapi, pada kasus ini kita dapat menggunakan v-if pada sebuah elemen(tag) dengan nama template, dimana elemen template ini akan bertindak sebagai pembungkus atau mengelompokkan beberapa elemen sekaligus. Dengan menggunakan sintaks template ini, maka sintaks template ini tidak akan terlihat didalam DOM. Intinya sintaks template ini hanya pengganti sebagai pembungkus untuk melakukan Confitional Rendering.
